# Ajustes finais da tela Mercado

Este pacote aplica os seguintes ajustes na tela **Mercado**:

- atualização automática dos gráficos a cada **1 minuto**
- renomeação do campo para **Intervalo do gráfico**
- exibição dinâmica do rótulo **Variação de X**, onde **X** é o intervalo selecionado
- ordenação alfabética dos cards/gráficos por símbolo

## Arquivo alterado

- `frontend/src/pages/MercadoPage.jsx`
