# Ajuste final da tela Mercado

Este overlay corrige apenas os rótulos de resumo no topo da tela Mercado para remover vírgulas indevidas na exibição:

- `Pares {base}: {quantidade}`
- `Selecionados: {quantidade}`
- `Favoritos: {quantidade}`

Arquivos incluídos:

- `frontend/src/pages/MercadoPage.jsx`
- `docs/MERCADO-AJUSTE-ROTULOS.md`

Aplicação:

1. Extraia o ZIP na raiz do projeto.
2. Aceite sobrescrever os arquivos.
3. Rode o build/deploy do frontend.
