Aplicação do overlay

1. Extraia este ZIP na raiz do projeto.
2. Aceite sobrescrever os arquivos.
3. Faça o build/deploy normal do frontend.

O que este overlay ajusta
- abre Operações, Execução e Social em modal filtrado por moeda
- sincroniza Favoritos <-> Configuração > Símbolos
- mantém favoritos visíveis nos cards mesmo trocando a Base de conversão
- deixa a estrela amarela quando a moeda está favoritada
- remove o limite de pares selecionados na lista
