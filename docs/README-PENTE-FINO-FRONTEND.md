# Pente-fino frontend

Pacote focado em acabamento visual do painel.

## O que foi ajustado
- grid e equalização visual dos cards do dashboard
- scroll interno em tabelas e listas dentro dos cards
- sidebar com rolagem própria
- padronização visual de inputs, selects e focos
- definição completa dos estilos da página Mercado
- mini gráficos (`sparkline`) com aparência consistente
- estados visuais de favoritos, presets, tags e comparativos
- refinamento responsivo para telas menores

## Aplicação
Copie o arquivo abaixo por cima do projeto:
- `frontend/src/styles.css`

Depois rode o build/deploy normal do frontend.
